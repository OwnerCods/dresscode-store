# 🎉 Первый запуск - Пошаговая инструкция

## Шаг 1: Проверка установки Node.js

```bash
node --version  # Должно быть >= 18.0.0
npm --version   # Должно быть >= 9.0.0
```

Если Node.js не установлен, скачайте с https://nodejs.org/

## Шаг 2: Установка зависимостей

### Backend
```bash
cd E:\telegaMiniAppDresscode\server
npm install
```

Ожидайте завершения установки (может занять 1-2 минуты).

### Frontend
```bash
cd E:\telegaMiniAppDresscode\tg
npm install
```

## Шаг 3: Настройка Backend

### 3.1 Откройте файл `server/.env`
Файл уже создан, нужно заполнить данные:

```env
PORT=3001
NODE_ENV=development

# YooMoney - получите на https://yookassa.ru/
YOOMONEY_SHOP_ID=your_shop_id_here
YOOMONEY_SECRET_KEY=your_secret_key_here

# URL вашего бота (пока можно оставить как есть)
FRONTEND_URL=https://t.me/your_bot_name

# Ваш Telegram ID (узнайте у @userinfobot)
ADMIN_TELEGRAM_IDS=123456789

DATA_DIR=./data
```

### 3.2 Узнайте свой Telegram ID
1. Откройте Telegram
2. Найдите бота @userinfobot
3. Отправьте ему любое сообщение
4. Скопируйте ваш ID
5. Вставьте в `ADMIN_TELEGRAM_IDS`

## Шаг 4: Настройка Frontend

### 4.1 Откройте файл `tg/.env.local`
```env
VITE_API_URL=http://localhost:3001/api
VITE_ADMIN_IDS=123456789  # Ваш Telegram ID
```

## Шаг 5: Первый запуск (без YooMoney)

Можно запустить проект БЕЗ настройки YooMoney для тестирования интерфейса.

### 5.1 Запустите Backend
Откройте первый терминал:
```bash
cd E:\telegaMiniAppDresscode\server
npm run dev
```

Вы должны увидеть:
```
🚀 Server is running on port 3001
📍 API URL: http://localhost:3001
🏥 Health check: http://localhost:3001/health
```

### 5.2 Запустите Frontend
Откройте второй терминал:
```bash
cd E:\telegaMiniAppDresscode\tg
npm run dev
```

Вы должны увидеть:
```
VITE v7.x.x  ready in xxx ms

➜  Local:   http://localhost:3000/
➜  Network: use --host to expose
```

### 5.3 Откройте браузер
Перейдите на http://localhost:3000

Вы должны увидеть:
- ✅ Каталог товаров
- ✅ Возможность добавить в корзину
- ✅ Корзину с товарами
- ✅ Вкладку "Заказы"
- ✅ Вкладку "Админ" (если указали свой ID)

## Шаг 6: Тестирование без оплаты

1. Добавьте товары в корзину
2. Перейдите в корзину
3. Нажмите "Перейти к оформлению"
4. Заполните телефон (обязательно)
5. Нажмите "Перейти к оплате"

**Важно:** Без настройки YooMoney оплата не будет работать, но заказ создастся в системе.

## Шаг 7: Проверка админ-панели

1. Перейдите на вкладку "Админ"
2. Вы должны увидеть:
   - Статистику заказов
   - Список всех заказов
   - Возможность менять статусы

## Шаг 8: Настройка YooMoney (для реальных платежей)

### 8.1 Регистрация
1. Перейдите на https://yookassa.ru/
2. Нажмите "Подключить ЮKassa"
3. Заполните данные (нужны данные ИП или ООО)
4. Дождитесь активации

### 8.2 Получение ключей
1. Войдите в личный кабинет YooKassa
2. Перейдите в "Настройки" → "Интеграция"
3. Скопируйте:
   - **shopId** (идентификатор магазина)
   - **Секретный ключ**

### 8.3 Добавьте ключи в server/.env
```env
YOOMONEY_SHOP_ID=123456  # Ваш реальный shopId
YOOMONEY_SECRET_KEY=live_ваш_секретный_ключ
```

### 8.4 Перезапустите Backend
```bash
# Остановите сервер (Ctrl+C)
# Запустите снова
npm run dev
```

### 8.5 Тестовые платежи
YooKassa предоставляет тестовый режим:

**Тестовая карта (успешная оплата):**
- Номер: `5555 5555 5555 4477`
- Срок: любая будущая дата (например, 12/27)
- CVC: любые 3 цифры (например, 123)

**Тестовая карта (отклонение):**
- Номер: `5555 5555 5555 5599`

## Шаг 9: Создание Telegram бота

### 9.1 Создайте бота
1. Найдите @BotFather в Telegram
2. Отправьте `/newbot`
3. Придумайте имя (например, "Dress Code Store")
4. Придумайте username (например, "dresscode_store_bot")
5. Сохраните токен бота

### 9.2 Создайте Mini App
1. Отправьте @BotFather команду `/newapp`
2. Выберите вашего бота
3. Укажите название приложения
4. Загрузите иконку (512x512 px)
5. Укажите URL: `http://localhost:3000` (для теста)

### 9.3 Тестирование в Telegram (локально)
Для локального тестирования используйте ngrok:

```bash
# Установите ngrok: https://ngrok.com/
ngrok http 3000
```

Вы получите публичный URL (например, `https://abc123.ngrok.io`)

Обновите URL Mini App в @BotFather на этот адрес.

## 🆘 Частые проблемы

### Backend не запускается
```bash
# Проверьте, что порт 3001 свободен
netstat -ano | findstr :3001

# Если занят, измените PORT в server/.env
```

### Frontend не подключается к Backend
```bash
# Проверьте VITE_API_URL в tg/.env.local
# Должно быть: http://localhost:3001/api
```

### Ошибка "Cannot find module"
```bash
# Переустановите зависимости
rm -rf node_modules package-lock.json
npm install
```

### Админ-панель не отображается
```bash
# Проверьте, что ваш Telegram ID указан в:
# 1. server/.env → ADMIN_TELEGRAM_IDS
# 2. tg/.env.local → VITE_ADMIN_IDS
```
