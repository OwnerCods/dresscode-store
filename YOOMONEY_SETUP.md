# Настройка YooMoney (YooKassa)

## 1. Регистрация

1. Перейдите на https://yookassa.ru/
2. Нажмите "Подключить ЮKassa"
3. Заполните данные компании/ИП
4. Дождитесь активации аккаунта

## 2. Создание магазина

1. Войдите в личный кабинет
2. Перейдите в раздел "Магазины"
3. Нажмите "Создать магазин"
4. Заполните информацию о магазине

## 3. Получение ключей

1. В личном кабинете откройте настройки магазина
2. Перейдите в раздел "Интеграция"
3. Скопируйте:
   - **shopId** (идентификатор магазина)
   - **Секретный ключ** (secret key)

## 4. Настройка в проекте

Добавьте ключи в `server/.env`:

```env
YOOMONEY_SHOP_ID=123456
YOOMONEY_SECRET_KEY=live_ваш_секретный_ключ
```

## 5. Настройка Webhook

1. В личном кабинете YooKassa перейдите в "Уведомления"
2. Добавьте URL webhook:
   ```
   https://your-domain.com/api/payment/webhook
   ```
3. Выберите события:
   - payment.succeeded
   - payment.canceled

## 6. Тестовый режим

Для тестирования используйте тестовые ключи:

```env
YOOMONEY_SHOP_ID=test_shop_id
YOOMONEY_SECRET_KEY=test_secret_key
```

### Тестовые карты:

**Успешная оплата:**
- Номер: `5555 5555 5555 4477`
- Срок: любая будущая дата
- CVC: любые 3 цифры

**Отклоненная оплата:**
- Номер: `5555 5555 5555 5599`

## 7. Комиссии

- Стандартная комиссия: 2.8% + 10₽
- Минимальная сумма платежа: 1₽
- Максимальная сумма: 500,000₽

## 8. Проверка интеграции

1. Запустите сервер
2. Создайте тестовый заказ
3. Проверьте логи сервера
4. Убедитесь, что платеж создается в YooKassa
